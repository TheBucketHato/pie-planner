# Pie Planner

A tiny web app to plan your day visually using pie-chart time slices.

## Features
- Visual time-slice planning using a pie-style layout
- Quick add/edit of tasks with duration
- Minimal, dependency-free frontend (HTML/CSS/JS)
 - Built-in dark mode toggle with persistent preference

## Files
- `index.html` — app entry and UI
- `styles.css` — styling
- `app.js` — application logic
- `README.md` — this file

## Usage
1. Open `index.html` in your browser.
2. Add tasks and durations to build your day's pie.

## Development
- Edit `app.js` and `styles.css` and reload the page to see changes.
- No build step required; works with a static file server or directly from the filesystem.
